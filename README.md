# 동아리 모집 정보 확인 서비스

[서비스 바로가기](https://skhu.eoehd1ek.com)

교내 동아리 모집 정보를 검색하여 찾아볼 수 있는 서비스입니다.

동아리 모집 정보는 교내 커뮤니티에 올라온 정보를 사용했습니다.

동아리 모집 정보의 추가, 수정, 삭제 요청은 에브리타임 게시글을 통한 쪽지로 요청바랍니다.

## 설계

- 단기 사용 목적으로 빠른 제작을 위해 단일 파일로 작성되었습니다.
- 이미지는 /image 하위 경로에 존재합니다.  
- 동아리 모집 정보글의 본문은 /contents 하위 경로에 존재합니다.  
- 상세 페이지는 data.json의 keyword를 사용하여 이미지와 본문 정보를 가져옵니다.
